# Changelog

## 0.1.0

- Initial release of the Valenty workspace
- **valenty_dsl**: Compile-time safe Gherkin-style DSL with phantom types, builder base classes, ScenarioRunner, multi-channel testing, fixtures, and TestContext
- **valenty_cli**: AI-powered CLI with `init`, `generate skills`, `scaffold feature`, `list`, `context`, `validate`, `test`, `doctor`, and `update` commands
